# 冒泡排序
有4种优化手段，分别是
- 记录是否已有序
- 记录是否已有序 + 记录有序区边界
- 记录是否已有序 + 左右来回交换
- 记录是否已有序 + 左右来回交换 + 记录有序区边界

![](../../pic/Snipaste_2018-08-20_21-09-32.jpg)

其中左右来回交换又称为鸡尾酒排序
